Her Rhythum

About the Project

Her Rhythm is an Android mobile application developed as part of the Mobile Application Development paper at Massey University. The concept and overall design were created by myself in collaboration with another student.
The project was inspired by research showing that menstrual cycles can influence factors such as energy levels, exercise performance, and body weight. Rather than focusing on population-wide trends, the goal was to explore how a mobile application could help individual users better understand the relationship between their own cycle and their personal health data.

Application Overview

One of the ideas that inspired this project was the observation that women are often told that their menstrual cycle affects things like energy levels, exercise performance, and weight fluctuations. While there is research supporting many of these claims, every individual is different, and population-level trends do not always reflect personal experience.
Her Rhythm was created as a way for users to explore those relationships for themselves. By combining menstrual cycle, weight, and exercise data, the application allows users to build their own dataset and look for patterns over time. Rather than drawing conclusions on the user's behalf, the goal is to provide tools that help users better understand how their own body responds throughout different stages of their cycle.
From a software development perspective, this project provided experience designing and implementing a complete Android application using modern development practices. The project involved creating an intuitive user interface, managing persistent user data, implementing cycle-based calculations, and developing custom data visualisations to present information in a meaningful and accessible way. Particular attention was given to creating a clear user experience, ensuring data remained consistent across the application, and presenting complex information in a format that encouraged exploration and understanding.

Application Screens

  • Home Screen

    • Shows the user's current cycle, predicted dates, and recent weight trend.
    • Provides quick access to the main tracking and graph screens.

  • Add Period Screen

    • Allows users to record whether they were bleeding or not bleeding on a selected date.
    • Updates cycle calculations and handles changes to existing entries.

  • Add Weight Screen

    • Allows users to record a weight value for a selected date.
    • Updates stored data so weight graphs reflect new entries.

  • Add Exercise Screen

    • Allows users to log exercise machine, sets, reps, weight, and date.
    • Stores exercise records for later progress tracking and graphing.

  • View Weight Info Screen

    • Displays weight trends graphically, either by cycle or across a custom date range.
    • Uses cycle phase bands to show weight data in menstrual-cycle context.

  • View Exercise Info Screen

    • Displays exercise progress over time for a selected machine.
    • Uses total load to represent training effort across logged sets, reps, and weights.

  • Top Bar

    • Shows the current page title and provides dropdown navigation.
    • Includes access to the debug screen for testing and demonstration purposes.

  • DEBUG & Demo Data Screen

    • Provides tools for inserting sample period, weight, and exercise data.
    • Includes options to clear stored data and demonstrate graph features quickly.

Features

  • Menstrual Cycle Tracking

    Users can record bleeding days, allowing the application to identify menstrual cycles, calculate cycle lengths, and estimate future cycle information.

  • Weight Tracking & Analysis

    Users can record weight measurements over time and visualise trends both within individual cycles and across custom date ranges.

  • Exercise Tracking

    Users can log exercise sessions, including machine used, sets, repetitions, and weight lifted, creating a historical record of training performance.

  • Cycle-Aware Data Visualisation

    Weight data is displayed alongside menstrual cycle phases, helping users explore potential relationships between their cycle and health metrics.

  • Exercise Performance Monitoring

    Exercise records can be viewed over time and filtered by machine, allowing users to track changes in performance and training load.

  • Custom Date Range Reporting

    Users can generate graphs for selected date ranges rather than being restricted to predefined cycle periods.

  • Local Data Storage

    All user-entered period, weight, and exercise data is stored locally on the device using a Room database.
