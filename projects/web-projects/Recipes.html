About the Project

This project was done as part of exploring The Odin Project to continue learning and practicing skills. It specifically was about practicing using HTML to build a basic website consisting of a few pages, basic navigation, use of links and images, using Git and GitHub and finally publishing the page to GitHub Pages.

Features

  • Home Page

    A simple homepage (index.html) that contains links to several recipes.

  • Recipes

    Several additional Recipe pages are included.
    Each recipe page includes:

    • a title,
    • an image (with attribution),
    • a description,
    • a list of ingredients (unnordered),
    • a list of instructions (ordered)
    • a back to home link

  • Published GitHub Page

    The Project was published to GitHub pages and can be found at this address: https://pixellette.github.io/OdinProj-Recipes/
