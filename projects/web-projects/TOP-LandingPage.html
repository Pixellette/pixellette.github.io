About the Project

This project was done as part of exploring The Odin Project to continue learning and practicing skills. It specifically was about practicing using CSS to build a basic website layout using Flexbox. We were told not to worry about making it cope with different sizes, but that we were welcome to customise it.
I chose to customise mine by making it about my cats.
